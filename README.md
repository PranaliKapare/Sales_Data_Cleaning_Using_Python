# Sales Data Cleaning Using Python

## Project Overview
This project demonstrates cleaning and preprocessing of sales data using **Python (Pandas & NumPy)**.  
The goal is to handle missing values, duplicates, inconsistent formatting, and outliers to create a clean dataset ready for analysis.

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- CSV

---

## Key Steps
1. Load the raw dataset using Pandas.
2. Handle missing values in numeric and categorical columns.
3. Remove duplicate records.
4. Standardize date and string formats.
5. Detect and handle outliers.
6. Export the cleaned dataset as a CSV file.

---

## Outcome
- A **clean and structured dataset** ready for analytics or data pipelines.
